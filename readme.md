# App de Autenticação – React Native (Expo) | MVVM

## Este projeto é um aplicativo simples de Registro e Login, desenvolvido com React Native + Expo, seguindo a arquitetura MVVM (Model–View–ViewModel). O objetivo é demonstrar uma base sólida e escalável para aplicativos mobile modernos com consumo de API, gerenciamento de estado e estilização otimizada.

### Tecnologias Utilizadas

- Expo

- React Native

- React Query

- Axios

- NativeWind

- Tailwind Variants

- MVVM Architecture

### Arquitetura (MVVM)

O projeto segue a estrutura MVVM, garantindo organização, testabilidade e escalabilidade

### Funcionalidades

- Criar conta

- Realizar login

- Validação de formulário com zod e React Hook Form

- Persistência do usuário autenticado

- Separação clara entre UI, lógica e regras de negócio
